# 3D Simulation of ASTU GEDA Gate

<img width="1280" height="720" alt="image" src="https://github.com/user-attachments/assets/49fe7765-bd2c-4db6-bf52-e4cb75cf40f6" />

This project is a complete 3D simulation of the ASTU GEDA Gate built from scratch using **C++ and OpenGL (FreeGLUT)**.

It started as a simple 2D class project, and later I expanded it into a full 3D environment through weeks of learning, improvement, and experimentation.
